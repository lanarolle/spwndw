# SpendWise

SpendWise is a lightweight expense tracker Android app written in Kotlin. It helps you track your expenses, categorize spending, and view simple summaries to better understand your finances.

## Features

- Add, edit, and delete expenses
- Categorize expenses (e.g., Food, Transport, Bills)
- View expense lists and simple summaries
- Local persistence using Room (Kotlin)
- Clean, minimal UI optimized for quick entry

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

- Android Studio (Electric Eel or newer recommended)
- JDK 11 or later
- Gradle (bundled with Android Studio)

### Build & Run

1. Clone the repository:

   git clone https://github.com/lanarolle/spwndw.git
2. Open the project in Android Studio.
3. Let Gradle sync and build the project.
4. Run the app on an emulator or an Android device.

## Project Structure

- app/ - Android app module
- data/ - Data layer (Room, repositories)
- ui/ - Activities/Fragments and ViewModels
